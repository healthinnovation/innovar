# Peru names and spatial

A dataset containing the names of the 1874 districts, 196 provinces and
25 departments of Peru according INEI - 2017, additionally also is the
limits administrative level 3 in a spatial vector format.

## Usage

``` r
data(Peru)
```

## Format

A dataset with 1874 rows and 9 variables:

- **ubigeo**:

  Code of six numbers for each district

- **dep.code**:

  Code to each departament

- **dep**:

  Name of each departament

- **prov.code**:

  Code to each province

- **prov**:

  Name of each province

- **distr.code**:

  Code to each district

- **distr**:

  Name of each district

- **capital**:

  Name of the capital main
