# Use Innovar Rmarkdown Flatly

This function populates the working directory with files corresponding
to the Innovar Rmarkdown Flatly.

## Usage

``` r
use_rmd_flatly(file)
```

## Arguments

- file:

  Name of the \`.Rmd\` file to create.

## Examples

``` r
if (FALSE) { # \dontrun{
library(innovar)
use_rmd_flatly("report_01.Rmd")
} # }
```
