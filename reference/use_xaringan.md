# Use Innovar Xaringan Template

This function populates the working directory with files corresponding
to the xaringan template of the package.

## Usage

``` r
use_xaringan(file)
```

## Arguments

- file:

  Name of the \`.Rmd\` file to create.

## Examples

``` r
if (FALSE) { # \dontrun{
library(innovar)
use_xaringan("class_01.Rmd")
} # }
```
